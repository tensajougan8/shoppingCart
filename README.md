# shoppingCart
Prequitse
- nodeJS
- TypeScript
- Docker
- Mongo
- Swagger
## Info
This is a shopping cart with user authentication add to cart and others

## Running docker
```bash
$ docker-compose up -d
```

## Running Project
```
npm i
npm run build
npm run start 
```

## Run test
```
npm run test
```

## Documentation
